COMMAND SYNTAX

(Please do not include these when typing commands)

 []

- Optional paramater

 <>

- Required parameter

 < thing1 || thing2 >

- Select an option inside

Get information on a command.

Usage: 

/help

Example:

/help

Get information on a user.

Usage: 

/user-info [user]

Example:

/user-info @Cytech#0001

Ban a user from your server!

Usage: 

/ban < member >

Example: 

/ban @Mee6#4876

Required Permissions: 

Ban Members

kick a user from your server!

Usage: 

/kick < member >

Example: 

/kick @Mee6#4876

Required Permissions: 

Kick Members

Purge/ delete a group of messages! (max of 100 messages)

Usage: 

/purge < amount >

Example: 

/purge 100

Required Permissions: 

Manage Messages
